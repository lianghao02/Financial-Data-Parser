# CSV 轉 Excel 金融資料轉檔工具 v1.6.0 (CSV to Excel Pro)

這是一個專為金融帳戶資料設計的網頁端轉換工具，能夠將 CSV 檔案快速轉換為格式正確的 Excel 檔案。特別針對台灣金融資料常見的格式問題（如帳號前導零、金額符號）進行了優化處理。

## ✨ 功能特色

- **直覺的操作介面**：支援拖放 (Drag & Drop) 檔案，亦可點擊選取。
- **批次與壓縮檔支援 (New v1.4)**：支援單選、多選 `.csv` / `.xlsx` / `.xls` 甚至 **資料夾** 與 **`.zip`**。工具會自動在前端遞迴展開所有的資料夾層級及解壓縮 ZIP。
- **三種匯出模式**：
  - **個別儲存**：將每個檔案轉換為獨立的 Excel 檔案，保留原始檔名。
  - **分頁合併**：將所有檔案合併至同一個 Excel 檔案的不同分頁。
  - **單頁合併** (New v1.2)：將多個檔案的資料「垂直堆疊」至同一 Sheet (自動去除重複標題)。
- **智慧歸戶面板 (New v1.5)**：
  - **自動辨識**：偵測開戶資料，產生垂直排版的個人資訊面板 (Dashboard)。
  - **視覺強化**：自動為每位對象的資料區塊繪製 **黑色粗外框線**，提升列印與閱讀質感。
  - **歸戶整理**：自動將分散在各檔的帳戶彙整至對應對象下方。
- **智慧分頁命名 (New v1.6)**：
  - **兩階段掃描**：先預掃描開戶資料建立「身分證 ↔ 姓名」對照表，自動將只有身分證號的分頁重新命名為正確的「戶名」。
- **UI 配置面板 (New v1.3)**：
  - **資料格式**：可選擇「純文字」(保留首位 0)、「數值格式」或「系統自動辨識」。
  - **清除貨幣符號**：自動移除 `$`, `+`, `,` 及 `.00` 尾碼。
  - **匯出檔名設定**：自訂輸出檔案名稱。
  - **分頁名稱前綴**：合併模式下可自訂分頁開頭。
- **技術升級 (New v1.6)**：
  - **xlsx-js-style**：升級核心庫，支援在網頁端生成帶有框線樣式的專業 Excel。
- **防禦機制**：
  - **大檔案與壓縮檔處理**：針對 50MB+ 檔案或遞迴解壓縮提供載入遮罩。
  - **獨立錯誤處理**：單一檔案格式錯誤不影響同批次其他檔案的轉換進度。

## 🚀 使用方式

本專案採用 **單檔交付 (Single-File Distribution)** 架構，無需安裝任何環境。

1. **開啟**：直接使用瀏覽器開啟 `index.html`。
2. **操作**：
   - 設定右側/底部的轉換參數。
   - 拖曳 `.csv` 檔案至中央區域。
   - 點擊「開始轉換」。

> **注意**：本工具依賴 CDN 載入核心函式庫 (Tailwind CSS, SheetJS)，請確保您的裝置已連上網際網路。

## 🛠️ 技術棧

- **HTML5**: 單頁應用結構。
- **Tailwind CSS (CDN)**: 透過 CDN 即時渲染樣式。
- **JavaScript (Vanilla)**: 內嵌核心邏輯 (ES6+)。
- **SheetJS (CDN)**: 用於 Excel 生成。
- **SVG Icons**: 內嵌向量圖示，無外部圖檔請求。
